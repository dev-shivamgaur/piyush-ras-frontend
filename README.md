# Piyush Rash — Frontend

React (Vite) single-page app for browsing **Hindi kavitā**, viewing cards, managing uploads, and using dashboard/profile screens. Styling uses **Tailwind CSS v4**; routing uses **React Router v7**.

## Tech stack

- **React** 19 · **Vite** 8  
- **Tailwind CSS** 4 (`@tailwindcss/vite`)  
- **React Router DOM** 7  
- **React Hook Form** · **React Icons**

## Prerequisites

- **Node.js** (LTS recommended)  
- **npm** (or compatible package manager)

## Setup

```bash
cd Frontend
npm install
npm run dev
